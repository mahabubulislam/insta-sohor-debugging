Some JavaScript common question

Question 1: How does JavaScript work?

Answer:JavaScript হচ্ছে একটি interpreted ল্যাংগুয়েজ।এটি V8 engine এর সাহায্যে চলে। V8 engine একটি ওপেন সোর্স JavaScript engine.এটি C++ ল্যাংগুয়েজ দিয়ে তৈরি করা হয়েছে। এটি Chromium based broweser(Example: Google Chrome) ও Node JS এ ব্যবহার হয়েছে। Interpreted ল্যাংগুয়েজ রান হতে সময় লাগায় JavaScript আপডেট হয়ে এখন JIT(Just in time) Compiler এর সাহায্যে রান হয়। এর ফলে খুব দ্রুত কোড execute হয়।

Question 2: What does JavaScript event loop do?

Answer:JavaScript মুলত Event loop অনুসারে কাজ করে। আমরা জানি JavaScript একটি asynchronous programming language. জাভাস্ক্রিপট তার সব প্রোগ্রাম single threaded এর মাধ্যমে execute করে। Event loop কে সাধারনত জাভাস্ক্রিপট লুপের মতো ভাবা যেতে পারে। এক্ষেত্রে লুপ টা জাভাস্ক্রিপট নিজেই পরিচালনা করে। যখন কোড execute করা হয় তখন JavaScript দেখে তার stack এ কি কি টাস্ক আছে, প্রথমে সেগুলো execute করে তারপর শেষ হলে দেখে queue তে কি কি টাস্ক আছে সেগুলো execute করে । এভাবেই এটি লুপের মতো চলতে থাকে। Event loop এভাবেই কাজ করে।

Question 3:Differece between Local storage and Session storage

Answer: Browser এ দুটি storage থাকে। একটি হলো Local storage এবং আর একটি Session storage। এই দুটির মাঝে পার্থক্য হচ্ছেঃ-

  • Local Storage:
    • Local storage হচ্ছে যখন কোন user কোন browser থেকে কোন একটি ওয়েবসাইট ব্রাউজ করছে তখন ওই ওয়েবসাইটে তার কিছু ইনফরমেশন বা ডাটা সেভ হয়ে থাকে। এই ডাটা ওই Browser এর Local storage এ স্টোর হয়ে থাকে। এই ডাটা ওই ইউজার যতবার এই ব্রাউজার দিয়ে ওই ওয়েবসাইটে প্রবেশ করবে ততবার ই আগের ডাটা স্টোর হয়ে থাকে। এই ডাটা তখনি ডিলিট হয়ে যাবে যখন Browser টি remove বা uninstall করা হবে। উদাহরনঃ বিভিন্ন ই-কমার্স সাইটে লগইন করা ছাড়াই কোন items কার্ট এ যোগ করলে এটি Local storage এ সেভ হয়ে থাকে।

  • Session Storage:
    • Session storage হচ্ছে যখন কোন user একটি নির্দিষ্ট session বা সময় পর্যন্ত browser এ থাকে এবং কোন একটি ওয়েবসাইট ব্রাউজ করছে তখন ওই ওয়েবসাইটে তার কিছু ইনফরমেশন বা ডাটা সেভ হয়ে থাকে। এই ডাটা ওই ওয়েবসাইট বার বার রিফ্রেশ করলেও থেকে যাবে যতক্ষন পর্যন্ত browser টি বন্ধ করা না হবে। Browser টি পুনরায় চালু করলে পুর্বের সেশনের কোন ডাটা স্টোর হয়ে থাকবে না।
